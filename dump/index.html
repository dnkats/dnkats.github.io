FCIDump files

ElemCo.FciDump — Module

Read and write fcidump format integrals. Individual arrays of integrals can also be in *.npy format

source

The FCIDump file format is a simple text file format for storing molecular integrals. It is used by many quantum chemistry programs.

Storage of molecular integrals

ElemCo.FciDump.FDump — Type
FDump

Molecular integrals

The 2-e integrals are stored in the physicists' notation: int2[pqrs] = <pq|rs> and for triang the last two indices are stored as a single upper triangular index (r <= s)

  • int2: 2-e⁻ integrals for restricted orbitals fcidump. Default: []

  • int2aa: αα 2-e⁻ integrals for unrestricted orbitals fcidump. Default: []

  • int2bb: ββ 2-e⁻ integrals for unrestricted orbitals fcidump. Default: []

  • int2ab: αβ 2-e⁻ integrals for unrestricted orbitals fcidump. Default: []

  • int1: 1-e⁻ integrals for restricted orbitals fcidump. Default: []

  • int1a: α 1-e⁻ integrals for unrestricted orbitals fcidump. Default: []

  • int1b: β 1-e⁻ integrals for unrestricted orbitals fcidump. Default: []

  • int0: core energy Default: 0.0

  • head: header of fcidump file, a dictionary of arrays. Default: Dict()

  • triang: use an upper triangular index for last two indices of 2e⁻ integrals. Default: true

  • uhf: a convinience variable, has to coincide with head["IUHF"][1] > 0. Default: false
