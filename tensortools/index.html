ElemCo.TensorTools.ints2 — Function
ints2(EC::ECInfo, spaces::String, spincase = nothing, detri = true)

Return subset of 2e⁻ integrals according to spaces.

The spincase∈{:α,:β} can explicitly be given, or will be deduced from upper/lower case of spaces specification. If the last two indices are stored as triangular and detri - make them full, otherwise return as a triangular cut.
